# 商品同步助手 Chrome 插件

## 1. 启动本地同步服务

在项目根目录运行：

```bash
npm run sync-server
```

服务地址固定为：

```text
http://127.0.0.1:17321
```

插件不会直接保存飞书 token，而是把采集结果发给这个本地服务，由服务调用 `lark-cli` 写入飞书。

## 2. 安装插件

1. 打开 Chrome：`chrome://extensions/`
2. 打开右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本目录：`chrome-extension`

## 3. 简易使用

1. 打开 1688、淘宝、天猫、拼多多、Temu、SHEIN 的单个商品详情页
2. 点击浏览器右上角「商品同步助手」
3. 点击「采集并同步到飞书」

数据会写入：



后续按根目录的 `导入产品模板.xls` 导出：

```bash
npm run export-collected
```

## 4. 当前采集字段

字段直接按根目录 `导入产品模板.xls` 的表头：

- *产品主编号
- *产品名称
- 货币类型
- 产品主图
- 货源链接
- 货源平台
- 货源ID
- 详情描述
- 详情图
- 货源类目
- 自定义属性
- 产品视频
- 产品证书
- 尺寸图表
- SKU规格1
- SKU规格2
- 平台SKU
- *SKU售价
- SKU图片
- SKU库存
- SKU重量(KG)
- SKU尺寸(CM)

不同平台页面结构经常变，插件优先做通用采集。如果某个平台字段不准，后续可以针对该平台补专用选择器。
